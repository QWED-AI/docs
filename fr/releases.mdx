### Install

<CodeGroup>

```bash pip
pip install qwed==4.0.0
```

```bash docker
docker pull qwedai/qwed-verification:4.0.0
```

```bash npm
npm install @qwed-ai/sdk@1.0.0
```

```bash cargo
cargo add qwed@1.0.0
```

```bash go
go get github.com/QWED-AI/qwed-verification/sdk-go@v4.0.0
```

</CodeGroup>
