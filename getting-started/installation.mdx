Get QWED up and running in minutes.

## Python SDK

```bash
pip install qwed
```

After installing, run the setup wizard to configure your LLM provider:

```bash
qwed init
```

This walks you through selecting a provider (OpenAI, Anthropic, Ollama, or any OpenAI-compatible endpoint), entering your API key securely, and writing a `.env` file with the correct permissions. See the [CLI reference](/advanced/cli) for details.

### Requirements
- Python 3.10+
- Optional: Redis (for caching)

## TypeScript SDK

```bash
npm install @qwed-ai/sdk
# or
yarn add @qwed-ai/sdk
# or
pnpm add @qwed-ai/sdk
```

## Go SDK

```bash
go get github.com/qwed-ai/qwed-go
```

## Rust SDK

```toml
# Cargo.toml
[dependencies]
qwed = "1.0"
```

## Full Stack (Self-Hosted)

For running the complete QWED stack locally:

```bash
# Clone the repository
git clone https://github.com/QWED-AI/qwed-verification.git
cd qwed-verification

# Start infrastructure (Redis, Prometheus, Grafana, Jaeger)
docker-compose up -d

# Install dependencies
pip install -e .

# Run the API
python -m uvicorn qwed_new.api.main:app --reload
```

### Infrastructure URLs

| Service | URL |
|---------|-----|
| API | http://localhost:8000 |
| Grafana | http://localhost:3000 |
| Prometheus | http://localhost:9090 |
| Jaeger | http://localhost:16686 |
