# QWED Documentation

Official documentation for the [QWED Verification Protocol](https://qwedai.com).

## Overview

This repository uses [Mintlify](https://mintlify.com) to host the documentation for QWED. It covers guides, API references, SDK documentation, and advanced architecture details.

## Development

1. Install the Mintlify CLI:
   ```bash
   npm i -g mint
   ```

2. Run the development server:
   ```bash
   mint dev
   ```

3. Preview at `http://localhost:3000`.

## Deployment

Changes pushed to the `main` branch are automatically deployed to [docs.qwedai.com](https://docs.qwedai.com).
